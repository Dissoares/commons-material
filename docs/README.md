# Commons Material Docs

Este diretório centraliza decisões e guias da biblioteca para evolucao multiempresa.

## Estrutura
- `architecture/`: visao de arquitetura, modulos, fronteiras e dependencias.
- `adr/`: Architecture Decision Records (ADRs) com contexto, decisao e consequencias.
- `guides/`: guias de implementacao e consumo.
- `standards/`: padroes de codigo, acessibilidade, testes e versionamento.

## Arquivos iniciais recomendados
- `adr/0001-template.md`: template base para decisoes arquiteturais.
- `guides/component-evolution-checklist.md`: checklist de evolucao de componentes.
- `guides/enterprise-consumption.md`: padrao de consumo para herdar UX global sem CSS local.
- `standards/google-angular-standards.md`: baseline de qualidade alinhado a Angular/Google.
- `standards/contributing.md`: fluxo de contribuicao e requisitos de PR.
